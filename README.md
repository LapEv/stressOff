## Stress Off

Это музыкальное приложение с совмешением различных звуков и музыки с возможностью автономной работы.
На стадии переделки на Typescript и касстомные хуки

---

### Используемые технологии

:heavy_minus_sign: Frontend (admin Panel): `React`, `Typescript`, `MobX`, `Axios`, `React Roter Dom`, `SSR`
:heavy_minus_sign: Backend: `Node.js`, `Express`, `JWT`, `NodeMailer`, `Mail parser`  
:heavy_minus_sign: Database: `MongoDB`, `Mongoose`  
:heavy_minus_sign: Mobile: `React Native (Expo)`, `React Navigation`, `Redux`, `Expo-SQLite`  
:heavy_minus_sign: DevOps: `Docker`, `Nginx`  
:heavy_minus_sign: Инструменты: `Eslint`, `Prettier`, `Lefthook`
